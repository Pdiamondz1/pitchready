# Start here (no tech experience needed)

This is the plain-language guide. If words like "terminal," "clone," or "npm" make your eyes
glaze over — this page is for you. You won't need any of that.

## The one thing to understand first

The helpful "magic" in this template — the part that automatically organizes your stuff and
keeps improving itself — runs inside a free app called **Claude Code**. It does **not** run
inside the ChatGPT website or the regular Claude chat website. Those are great for *talking*,
but they can't reach into your computer to set things up. **Claude Code can.**

The good news: you still just **talk in plain English.** Claude Code does all the techie steps
for you — downloading, setup, file organizing. You never type computer commands. Think of it
as a helper that can actually do things on your computer while you tell it what you want.

## Step 1 — Install Claude Code (one time, about 5 minutes)

Download the **Claude Code desktop app** from Anthropic and sign in with your Claude account.
It installs like any normal app — point and click. This is the only setup you do yourself.

## Step 2 — Tell it to start your project

Open Claude Code and paste this in, changing the last line to describe your project:

> Hi! I want to start a new project using the template at
> https://github.com/Pdiamondz1/hma_project_foundation
> I'm not very technical, so please do all the setup for me, make a fresh copy on my
> computer, then run the **setup-project** skill and walk me through the questions one at a
> time. My project is about: **[describe your project in one sentence]**

It will download a fresh copy and start asking you simple questions.

## Step 3 — Answer a few easy questions

It asks everyday things like *"What should we call it?"* and *"What's it for?"* — no tech
knowledge needed. If it mentions **"tiers"** or **"API keys,"** just say:

> Start with the simplest option, no keys.

(That's the free, zero-setup mode. It works completely on its own.)

## Step 4 — Use it by just talking

From now on, you add things by talking normally:

- *"Save this note about my idea…"*
- *"Add this file to the project."*
- *"What did I save last week about ___?"*

It quietly files everything away and keeps a tidy, searchable index for you.

## What to ignore for now (so you don't get overwhelmed)

You can **skip all of this** until you actually want it — none of it is required to start:

- The "console" / web dashboard
- API keys, "embeddings," Supabase, scheduling
- GitHub accounts and saving online

Start with **just talking to Claude Code.** Everything else is a "someday, if I want it"
upgrade — and you can ask Claude Code to set any of it up for you later, in plain English.

## If you get stuck

Just tell Claude Code, in your own words: *"I'm stuck — can you explain that more simply?"* or
*"Something went wrong, can you fix it?"* It's designed to help you through it.
